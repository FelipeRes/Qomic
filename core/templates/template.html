Sobre QOMIC


Criando quadrinhos digitais

Qomic é um sistema de publicação de quadrinhos digitais de teste para a matéria de Programação Para Internert do curso de Análise e Desenvolvimento de Sistemas do IFPI Campus Teresina Central. O sistema é apenas um protótipo para avaliar e entender os recursos do framework Django e a plataforma Heroku
